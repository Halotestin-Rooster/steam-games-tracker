About This App

This Steam Games Tracker is a simple interactive site that lets users sign up, log in, or browse as a guest. Users with an account can connect their Steam API credentials to load owned games and calculate a rating based on game review scores.

Features

  • Local authentication using browser local storage.
  • Guest browsing mode with limited access (no score calculation).
  • Steam API integration for owned games and cover art.
  • Search the library and select games to evaluate.
  • Home, tracker, and about pages for a clean site structure.

How it works

When you sign up, your account details are saved locally in your browser. Once logged in, you can enter your Steam API Key and Steam ID to fetch games from Steam. Guests can see game cards but cannot calculate ratings until they register.
